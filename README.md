# Crypt-Messenger
simple file client-server encryption/decryption messaging application.

In this application, swing is used for GUI and socket-programming is used for the communication.

# How To Run
Server Terminal

    -- javac *.java
          
    -- java Server
                   
Client Terminals

    -- java Client
